# Nightfall Game

This is a React application built with Vite, Firebase, and Tailwind CSS.

## Setup

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Locally**:
    ```bash
    npm run dev
    ```
    Open the link (e.g., `http://localhost:5173`) in your browser.

## Configuration

The Firebase configuration is located in `src/firebase.js`.

## Build

To build for production:
```bash
npm run build
```
